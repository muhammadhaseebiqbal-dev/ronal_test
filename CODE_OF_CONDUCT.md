# Code of Conduct

## Our Commitment
We are committed to a respectful, professional, and inclusive collaboration environment for everyone contributing to this project.

## Expected Behavior
1. Communicate respectfully and directly.
2. Focus feedback on code, behavior, and outcomes.
3. Assume good intent and ask clarifying questions before escalating.
4. Share context and evidence when reporting issues.
5. Accept and act on constructive review feedback.

## Unacceptable Behavior
1. Harassment, threats, or discriminatory language.
2. Personal attacks or hostile review behavior.
3. Publishing private information without consent.
4. Disruptive conduct that blocks collaboration.

## Enforcement
Project maintainers may remove content, request revisions, or restrict participation for violations of this policy.

## Reporting
Report conduct concerns to project maintainers through the repository communication channel used for the engagement.

## Attribution
This document is adapted from common open-source code-of-conduct conventions and simplified for this project.
