## Reporting a Vulnerability
If you discover a security issue, do not open a public issue. Please email the project maintainer with:
- A clear description of the issue.
- Steps to reproduce.
- Any relevant logs or screenshots.

We will acknowledge receipt within 72 hours and provide a remediation plan as soon as possible.

## Scope
This policy covers the application code, configuration, and deployment guidance in this repository.

## Best Practices
- Do not commit real API keys or App IDs.
- Use environment variables for secrets and production configuration.
- Keep dependencies up to date.
